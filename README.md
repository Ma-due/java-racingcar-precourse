## 요구사항 분석
---

1. 각 자동차의 이름을 입력 받는다.
	* 자동차의 이름은 쉼표(,)로 구분한다.
	* 검증 사항
		* 자동차의 이름은 5자 이하만 가능하다.

2. 사용자로부터 시도할 회수를 입력 받는다.
	* 검증 사항
		* 회수는 숫자만 입력 가능하다.

3. 각 차수별 이동 상태를 출력한다.
	* 이동은 각 차량별 random 값을 통해 4 이상이면 전진, 3 이하면 멈춘다.
	* 각 차수의 실행 결과는 아래와 같다.
		pobi : --
		woni : ----
		jun : ---

4. 우승자를 가린다.
	* 우승자는 가장 많은 칸을 이동한 자동차 이다.
	* 우승자는 여러명일 수 있다.

* 예외 상황시 에러 문구는 아래와 같이 출력한다.
	* "[ERROR] ERROR MESSAGE"
